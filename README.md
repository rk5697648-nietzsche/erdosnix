# LDPC Decoder RTL Category Theory Model in Lean

This repository provides a formal, modular abstraction of an LDPC decoder hardware system using category theory in Lean.

## Structure

- `src/LDPC/Category.lean`: Main Lean file with the category-theoretic model.
- `lakefile.lean`, `lean-toolchain`: Lean project configuration files.

## How to Use

1. Clone the repository.
2. Open in VS Code or your Lean editor.
3. Run `lake build` to check the Lean code.
